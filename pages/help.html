Help Page - FAQ

Q: What is FreeBSD Online Man Page Editor?
A: It is an online website editor for FreeBSD man page which is still developing. The purpose of FreeBSD Online Man Page Editor is to simplify the editing process in FreeBSD (CURRENT branch) man page. Here is the GitHub repo URL - https://github.com/Wang-Yan-Hao/man_page_editor.
Q: How to use the online editor to change document?
A: Please follow the below step.
  1. Go to the page you want to edit.
  2. Change the source code in the left section. Click the "Patch Download" button to download the patch about your change.
  3. Click the "FreeBSD Bugzilla" button, going to FreeBSD Bugzilla to submit the patch.

Q: Is there a document about how to contribute to the man page?
A: Yes, the "Chapter 11. Manual Pages" in the book "FreeBSD Documentation Project Primer for New Contributors" (https://docs.freebsd.org/en/books/fdp-primer/manual-pages/.) may help you.
